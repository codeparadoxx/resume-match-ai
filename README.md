# ResumeMatch AI

ResumeMatch AI is an NLP-based application that computes semantic similarity between resumes and job descriptions using BERT.

##  Features
- BERT-based sentence embeddings
- Cosine similarity scoring
- Match percentage and match level classification
- FastAPI backend for inference
- Streamlit frontend for user interaction

##  Tech Stack
- Python
- Sentence-BERT
- FastAPI
- Streamlit
- Scikit-learn

##  Match Levels
- Excellent Match
- Good Match
- Average Match
- Poor Match
